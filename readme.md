# BikroyElectronics - Server Side

## Overview

This repository contains the server-side code for the BikroyElectronics e-commerce application. It provides APIs for managing orders, flash sales, coupons, cart , wishlist account management and various backend admin functionalities. The server is built using Node.js and Express and interacts with MongoDB for data storage and retrieval.

## Getting Started

To get started with the server-side code, follow these steps:


1. Clone the repository:

    ```
    git clone https://github.com/Mustafiz82/BikroyElectronics-server
    ```

2. Navigate to the project directory:

    ```
    cd BikroyElectronics-server
    ```

3. Install dependencies:

    ```git 
    npm install
    ```


4. Configure Environment Variables

    Create a `.env` file in the root directory with the following variables:

```bash

MONGO_URI="Your_MongoDB URI"
SECRET_KEY="Your_Secret_KEY"

```

5. Run the development server:

    ```
    npm start

    ```

6. Open your browser and navigate to the provided port address to view the application.
